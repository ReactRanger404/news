# 🔍 开源情报系统

> 基于 PHP + JSON 的轻量级开源情报聚合系统
> 本系统仅用于学习和研究目的，请遵守相关法律法规

---

## 环境要求

- PHP 7.4 或更高版本（推荐 PHP 8.0+）
- PHP 扩展：curl、json、simplexml、mbstring
- 无需数据库

---

## 安装与部署

### 1. 安装 PHP

**Windows：**
1. 下载 PHP：https://windows.php.net/download/
2. 解压到 `C:\php\`
3. 将 `C:\php` 添加到系统 PATH 环境变量
4. 打开 `C:\php\php.ini`，确保以下扩展已启用：
   ```
   extension=curl
   extension=json
   extension=simplexml
   extension=mbstring
   ```

验证安装：
```bash
php --version
```

### 2. 启动服务

进入项目目录，使用 PHP 内置服务器：

```bash
cd 项目目录
php -S localhost:8080
```

### 3. 访问系统

- **新闻展示页**：http://localhost:8080/show.php
- **管理后台**：http://localhost:8080/source.php

## 配置说明

### config.json 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `admin_user` | 管理员账号 | admin |
| `admin_pwd` | 管理员密码（MD5） | admin的MD5 |
| `crawl_interval` | 爬取间隔（分钟） | **300（5小时）** |
| `max_news` | 最大活跃新闻数 | **3500** |
| `page_size` | 每页展示条数 | 80 |
| `request_timeout` | HTTP请求超时（秒） | 10 |
| `allow_industries` | 允许的行业分类列表 | [...] |
| `archive_enabled` | 是否启用旧新闻归档 | true |
| `archive_dir` | 归档目录 | archive |

## 核心功能

### 📡 智能爬取系统
- 每 **5 小时**自动执行一次全量爬取
- 同时支持 HTTP 和 RSS/Atom/HTML 解析
- 自动从原文提取 Open Graph 图片（有图率 79.5%）
- 数据源连通性自动检测

### 📦 新闻归档机制
- 活跃新闻超过 **3500 条**自动归档
- 旧新闻按 **发布日期** 存入 `archive/YYYY-MM-DD.json`
- 归档新闻可通过 **日期范围检索** 和 **关键词搜索** 重新找到
- 归档数据保留完整字段，支持全文搜索

### 🔥 24小时热点追踪
- 页面顶部实时显示过去 24 小时内各行业情报数量
- 自动提取热门关键词（基于标题分词）
- 点击行业标签或关键词可一键筛选

### ⭐ 收藏功能
- 点击新闻右上角的星标按钮收藏（使用浏览器 localStorage）
- 支持在弹窗中收藏/取消收藏
- 点击头部「我的收藏」查看所有已收藏新闻
- 收藏数据保存在本地浏览器，清除缓存会丢失

### 🔍 多维检索
- **关键词搜索**：标题 + 描述全文检索
- **行业筛选**：通过导航栏点击行业标签
- **时间窗口**：通过日期选择器指定起止日期
- **组合筛选**：关键词 + 行业 + 时间范围任意组合

### 🖼️ 智能图片展示
- 仅当新闻包含 **真实图片** 时才显示缩略图
- 无图新闻完全隐藏图片区域，保持界面简洁
- 有图新闻自动缩放至 160×110 缩略图

### 📊 数据源管理 (source.php)
- 按行业分组展示所有数据源
- 支持新增、编辑、删除（单条/批量）
- 支持批量启用/禁用
- 支持 URL 连通性测试（单条/批量）

## 使用说明

### 启动方式

**方式一：一键启动（推荐）**
```bash
start.bat
```
自动打开 Web 服务器和后台爬虫守护。

**方式二：手动分开启动**
```bash
# 终端1：Web 服务器
php -S localhost:8080

# 终端2：后台爬虫（每5小时自动爬取）
php background_crawler.php
```

### 爬取

**手动触发爬取（无需登录）：**
```bash
php crawler.php
```

**通过 Web 触发（需先登录管理后台）：**
访问 http://localhost:8080/crawler.php

**远程触发（云平台）：**
```bash
curl http://你的域名/cron.php?token=你的密钥
```

### 新闻展示 (show.php)

- 行业筛选：点击导航栏的行业标签
- 关键词搜索：在搜索框输入关键词
- 时间筛选：使用日期选择器指定起止日期
- 收藏：点击新闻右上角的 ⭐ 按钮
- 热点追踪：页面顶部显示 24h 热点统计
- 分页导航：底部和顶部均有分页器

## 目录结构

```
/news/
├── config.json            # 系统配置
├── sources.json           # 数据源配置
├── data.json              # 活跃新闻数据
├── common.php             # 公共函数库
├── source.php             # 数据源管理
├── crawler.php            # 爬取程序
├── show.php               # 新闻展示
├── background_crawler.php # 后台爬取守护脚本
├── cron.php               # 远程爬取触发器
├── start.bat              # 一键启动脚本
├── archive/               # 归档新闻目录（自动生成）
│   ├── 2026-01-01.json
│   └── ...
├── logs/
│   └── crawl.log          # 爬取日志
```

## 安全说明

- 所有 JSON 读写均带文件锁，防止并发写入损坏
- 所有输入均经过 XSS 过滤
- URL 格式校验和重复检测
- 错误捕获，不抛出 PHP 原生报错
- 随机 User-Agent，防止被反爬识别
